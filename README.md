# Terminal Emulator
  
Module to perform actions on a terminal emulator  

*Read this in other languages: [English](README.md), [Espa√±ol](README.es.md).*

## How to install this module
  
__Download__ and __install__ the content in 'modules' folder in Rocketbot path  



## Overview


1. Connect  
Connect to terminal

2. Disconnect  
Disconnect terminal

3. End Session  
Finish terminal session

4. Send Text  
Send text to Terminal

5. Send Key  
Send key to Terminal

6. Move Cursor  
Move cursor to specific position on Terminal

7. Get Text  
Get text from Terminal

8. Wait  
Wait for text on terminal by specific condition
